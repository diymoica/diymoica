# DIY Moi Ca

**Build it. Automate it. Own it.**

> Open-source Home Automation projects — made in France, shared with the world.

---

## English

**DIY Moi Ca** is a community project around DIY home automation with Home Assistant and ESP32.
Every project is **open-source, local, no cloud, no subscription**.
We build things that work — and share everything so you can build them too.

### Projects

| Project | Description | Status |
|---|---|---|
| 🧪 **[DmD — DIY my Dose](https://github.com/diymoica/DmDose)** | Universal smart dosing controller — 4 peristaltic pumps via ESP32 + HA | 🔧 v0.9.9 — In production |
| 💡 **DmL** | Smart LED controller for planted tanks, reef aquariums & grow lights | 📋 Planned |
| 🌡️ **DmC Monitor** | Multi-probe monitoring — temperature, pH, salinity, dissolved O2 | 📋 Planned |
| ⚗️ **DmP** | Automated pH regulation with dosing feedback loop | 📋 Planned |

> All projects work together or independently on the same Home Assistant instance.

### Philosophy

```
No cloud.         Your data stays at home.
No subscription.  Free. Forever.
Your hardware.    Any ESP32, any pump, any probe.
Your rules.       You decide everything.
```
