**Build it. Automate it. Own it.**

---

## Qui sommes-nous ? / Who are we?

**DIY Moi Ca** est un projet communautaire autour de l'automatisation DIY avec Home Assistant et ESP32.
Chaque projet est **open-source, local, sans cloud, sans abonnement**.

**DIY Moi Ca** is a community project around DIY home automation with Home Assistant and ESP32.
Every project is **open-source, local, no cloud, no subscription**.

---

## Projets / Projects

| Projet | Description | Statut |
|---|---|---|
| 🧪 **[ReefDose](https://github.com/diymoica/reefdose)** | Universal smart dosing controller — 4 peristaltic pumps via ESP32 + HA | 🔧 Pre-release v0.9.8 |
| 💡 **ReefLed** | Smart LED controller for planted tanks, reef aquariums & grow lights | 📋 Planned |
| 🌡️ **ReefControl** | Multi-probe monitoring — temperature, pH, salinity, dissolved O2 | 📋 Planned |
| ⚗️ **ReefPH** | Automated pH regulation with dosing feedback loop | 📋 Planned |

> Tous les projets fonctionnent ensemble ou indépendamment.
> All projects work together or independently.
